Build a full-featured cross-platform chat application named "BmGapshap", inspired by WhatsApp, and also design and integrate a custom app icon.

====================
APP FEATURES
====================

Authentication:
- User signup/login with phone/email and OTP
- Secure authentication and authorization
- Profile setup with name, bio, and profile picture (DP)

Chats:
- One-to-one private chats
- Group chats with admin controls
- Real-time messaging using WebSockets

Calls:
- Voice calling

- Status stories (text/image/video) auto-expire in 24 hours

- Updates/Channels section

Communities:

Other:

- Linked devices support
- Dark and light mode
- Settings (account, privacy, notifications, storage)
- Media gallery
- Chat backup to cloud
- QR code scan login
- Camera integration
- Favorites list

Admin:
- Basic admin panel to manage users and reports
